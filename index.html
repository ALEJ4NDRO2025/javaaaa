Bases de JavaScript

¿Qué es?

JavaScript es un lenguaje de programación interpretado y de alto nivel (1995, Brendan Eich). Nació para dar interactividad a la web, que antes era mayormente estática con HTML y CSS.

¿Cómo funciona?

Se ejecuta en motores del navegador (V8 en Chrome, SpiderMonkey en Firefox). El código se interpreta en tiempo real, sin compilación previa, lo que permite respuestas inmediatas.

¿Para qué sirve?

Se usa tanto en el frontend como en el backend. Permite responder a acciones del usuario, actualizar partes de la página sin recargar y crear experiencias dinámicas (menús, efectos, etc.).

Actividad

Ejercicio 1: Suma

Ejercicio 3: Identificador de Edad

Clasifica la edad según el valor proporcionado por el usuario.

Verifica si el número ingresado corresponde a un entero.

Ejercicio 5: Calculadora Básica

Ejercicio 6: Identificador de Pares

Determina si un número es par o impar.

Ejercicio 7: Identificador de Género

Ejemplo didáctico para leer una entrada de texto y evaluar una condición.

Ejercicio 8: Tablas de Multiplicar

Genera dinámicamente una tabla de multiplicar.

Ejercicio 9: Lista a Mayúsculas

Convierte una lista de palabras a mayúsculas.

Ejercicio 10: Botón Bienvenida

Muestra un mensaje de bienvenida al pulsar el botón.

Ejercicio 11: Registro Bienvenida

Captura nombre, edad y trabajo, y muestra un saludo personalizado.
